# Island Watcher v1.6.1 — Local Only

Chrome extension that:

- Reads public island data from Fortnite.GG.
- Stores favorites and statistics locally in Chrome.
- Refreshes whenever the popup opens.
- Refreshes manually with the refresh button.
- Refreshes automatically every 1, 2, 5, or 10 minutes.
- Shows Players right now, 24 hour peak, and All time peak.
- Sends local Chrome notifications for new peaks.
- Does not use Cloudflare, a PWA, a backend, accounts, tokens, or remote synchronization.

## Install

1. Extract the ZIP.
2. Open `chrome://extensions/`.
3. Enable Developer mode.
4. Click Load unpacked.
5. Select the extracted folder.

## Update without losing favorites

Replace the files in the existing extension folder, then click Reload in `chrome://extensions/`.
Do not remove the installed extension if you want to preserve local data.
