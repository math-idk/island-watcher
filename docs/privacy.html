Privacy Policy

Island Watcher

Monitor public Fortnite Creative island statistics while keeping your preferences and notification settings stored locally in Chrome.

No accounts No analytics No personal data Local storage
View on GitHub ↗ Install from Chrome Web Store
🔒

Privacy first

No personal data collection

Island Watcher does not collect, sell, transmit or store personal information on an external server.

  • ✓ No user account or login is required.
  • ✓ No analytics or advertising trackers are used.
  • ✓ No personal profile is created.
  • ✓ No information is sold to third parties.
📊

Public information

What Island Watcher reads

The extension accesses public Fortnite.GG island pages to retrieve publicly displayed statistics.

  • ✓ Island name and island code
  • ✓ Current player count
  • ✓ 24-hour player peak
  • ✓ All-time player peak
  • ✓ Public island thumbnail
💾

Chrome storage

Information stored locally

Island Watcher stores the following information inside the user's Chrome browser so the extension can preserve its settings.

Favorite islands Island codes selected by the user.
Statistics Previously retrieved public island statistics.
Settings Refresh interval and notification preferences.
Card order The user's chosen order for favorite islands.
ℹ
Local processing

This information remains inside Chrome and is not uploaded to an Island Watcher server.

🔔

Notifications

Peak alerts

Island Watcher can display local Chrome notifications when a monitored island reaches a new player peak.

  • ✓ Notifications are created on the user's device.
  • ✓ Notification preferences are stored locally.
  • ✓ No notification history is sent to an external server.
🌐

Permissions and access

What the extension does not access

Island Watcher uses the minimum permissions necessary to retrieve public island statistics, store preferences and schedule local updates.

  • × Epic Games or Fortnite account credentials
  • × Passwords, authentication codes or payment information
  • × Browsing history or unrelated websites
  • × Personal files stored on the user's computer
  • × Private Fortnite or Epic Games account information
  • × Location, health, financial or communication data
🧩

Chrome permissions

Why each permission is used

Storage Saves favorite islands, settings, card order and retrieved statistics locally.
Alarms Schedules automatic updates at the interval selected by the user.
Notifications Displays local notifications when monitored islands reach new peaks.
Offscreen document Processes public Fortnite.GG island-page content in the background.
Fortnite.GG access Retrieves publicly displayed island information from Fortnite.GG.
📄

Policy updates

Changes to this policy

This privacy policy may be updated if Island Watcher's features or data practices change. The latest version will remain available on this page.

Any future version that introduces external synchronization, accounts, analytics or additional data processing will require this policy to be updated before that version is released.
