Island Watcher Privacy Policy

Last updated: July 13, 2026

Island Watcher is a Chrome extension that allows users to monitor public Fortnite island statistics.

Data collection

Island Watcher does not collect, sell, transfer, or store personal data on an external server.

Favorite island codes, settings, statistics, and notification preferences are stored locally in the user's Chrome browser.

External requests

The extension accesses public pages on Fortnite.GG to retrieve island information such as current players, 24-hour peak, all-time peak, island name, and thumbnail.

The extension does not access Epic Games login credentials, cookies, passwords, browsing history, personal files, or private Fortnite account information.

Notifications

Island Watcher may display local Chrome notifications when a monitored island reaches a new peak. Notification processing happens locally on the user's device.

Data sharing

Island Watcher does not share user data with third parties.

Contact

Questions about this privacy policy can be submitted through the project's GitHub repository.
